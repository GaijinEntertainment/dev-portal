Logs

The Logs section provides access to service logs and tools for filtering, viewing, and exporting log data.

To open logs, select the corresponding menu item.

The service supports two log types:

  • Environment logs

  • Common logs

Log Viewer

The Log Viewer displays log records and provides controls for managing columns, filters, and data export.

Export Logs

To export log data to a CSV file, click the Export button.

Column Management

To change the column order, open the column settings window and drag columns to the required position.

To add a column, click the + Column button.

Select the required column names from the list.

To remove a column, click the Cross icon next to the column name.

To export log data to a CSV file, click the Export button.

Filters Management

To copy the current filters, click the Copy button.

To upload filters from a JSON file, click the Upload button.

Set JSON filters and click the Load button.

To reload logs and fetch the latest data, click Reload.

Note

Filters can be restored from a URL. If the address bar contains data after the resource parameter, filter values are automatically restored from it.

Time Range

The Time Range section allows you to define the time interval for log queries.

To select a time range, open the date range menu.

If data cannot be loaded for the selected time range, an error message is displayed.

Filters

Filters allow you to narrow down log records based on specific criteria.

Add New Filter

To add a new filter, click Add filter.

Select a filter name from the list.

Select an operator from the list.

Enter a value in the Value field.

Click Save to add the filter.

Disable Filter

To disable a filter, use the toggle switch.

Delete Filter

To remove a filter, click the Delete button.

Filter Groups

Filters can be grouped and enabled or disabled as a group.

To create a new group, click Add new group.

Enable or disable a group using the toggle switch.

To add a filter to a group, click Add filter within the group.

Filters can also be added directly from a log row using the filter icon.

Values that match active filters are highlighted in the log table.

Log Viewer Chart

The chart visualizes log distribution over time.

To apply a time range from the chart, select a range using the cursor.

To view logs for a specific chart value, click the corresponding data point.

Note

To return to the previous filter state, use the browser Back and Forward buttons.

Log Viewer Table

Log Details

To view detailed information for a log entry, click the Details button.

To copy a value to the clipboard, click the Copy button.

Pagination

Use the Previous and Next buttons to navigate between pages.

To jump to a specific page, enter the page number in the input field.

If the requested page cannot be loaded, an error message is displayed.

Column Width

To resize a column, hover over the column divider and drag it to the desired width.

Table Settings

To open the table settings menu, click the Table settings button.

The settings window displays the number of hidden columns.

Manage column visibility using the visibility selector.

Column visibility can also be changed directly from the table.

Note

At least one column must remain visible.

To reorder columns, drag them to the required position in the settings panel.
