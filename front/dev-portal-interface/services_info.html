Services Info

The Services Info section provides detailed information about available services, including access URLs and version details.

View Services Information

To view information about a services, select the corresponding item from the list.

The system displays a table containing detailed parameters.

Copy Service URL

To copy the service URL to the clipboard, click the Copy to buffer button.

Version Information

The bottom of the table displays the currently deployed versions of the UI and the server.
